# Clone index with submodules

## One Step Clone
```
git clone --recursive git://github.com/GreenSense/Index.git GreenSense/Index

cd GreenSense/Index
```

## Two Step Clone
```
git clone git://github.com/GreenSense/Index.git GreenSense/Index

cd GreenSense/Index

git submodule update --init --recursive
```
